# ESP DESIGN PRO - Analysis Suite 🚀

Bienvenido a la versión Beta de **ESP DESIGN PRO**. Este repositorio ha sido organizado profesionalmente para facilitar el mantenimiento y la escalabilidad del proyecto.

## 📂 Estructura del Proyecto

### 🏢 Raíz de Proyecto
- `ESP_STUDIO_LAUNCHER.bat`: Lanzador principal para Windows.
- `ESP_LAUNCHER.ps1`: Script de inicio (PowerShell) con soporte para sincronización en la nube.
- `services/`: Motores de sincronización y utilidades de soporte (Python + Datos).
- `app_unified/`: Directorio principal de la aplicación web (React + Vite).

### 🌐 Aplicación Principal (`app_unified/`)
- `src/`: Todo el código fuente de la aplicación (Componentes, Hooks, Workers).
- `public/`: Bases de datos (Excel), Modelos 3D, e Imágenes estáticas.
- `tools/`: Scripts de pre-procesamiento de datos y utilidades de desarrollo.
- `docs_and_data/`: Documentación técnica y respaldos de datos.

---

## 🛠️ Cómo Iniciar

1. Asegúrate de tener **Node.js** y **Python** instalados.
2. Ejecuta `ESP_STUDIO_LAUNCHER.bat` para iniciar el entorno completo.

> [!TIP]
> Si es la primera vez que inicias, el sistema descargará automáticamente las dependencias necesarias y sincronizará los datos desde el servidor en la nube.

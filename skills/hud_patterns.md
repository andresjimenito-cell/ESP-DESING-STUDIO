# 🎨 Skill: HUD UI Patterns

## Description
Guidelines for maintaining the "Zero Space Waste" high-density architecture and professional HUD aesthetic.

## Patterns
- **Glassmorphic Cards:** Use `backdrop-blur-md` with semi-transparent dark backgrounds.
- **Micro-nav:** Compact buttons with neon borders for phase switching.
- **High-Density Tables:** Minimal padding, font size 12px-13px, monospace for numerical data.
- **Sidebar Config:** Move all non-essential inputs to a retractable sidebar or popover.

## Color Palette
- **Primary:** Neon Cyan (`#00f2ff`)
- **Success:** Matrix Green (`#00ff41`)
- **Warning:** Industrial Orange (`#ff9100`)
- **Background:** Deep Space Blue/Black (`#0a0a0a`)
